[목표]
AI 기반 커리어 에이전트 플랫폼 ‘BAESH’의 MVP를 React로 구현해주세요. 디자인은 모바일 앱 Toss처럼 **심플하고 직관적인 스타일**을 적용합니다.

[디자인 스타일 가이드]
- 전체 배경: 흰색 (#FFFFFF)
- 포인트 컬러: 파란색 (#3182f6)
- 타이포그래피: Pretendard 또는 San-serif 계열
- 여백: 요소 간 여백 충분히 (padding 최소 p-4~p-6)
- 버튼: 라운드형, 그림자 있고 hover 시 부드럽게 강조
- 레이아웃: 모바일 우선 반응형 (PC 확장 고려)
- 애니메이션: Framer Motion으로 부드러운 탭 전환/로딩 효과 적용

[기술 스택]
- React (component 기반)
- Tailwind CSS
- Zustand 또는 Context API
- Framer Motion

[기능 구성요소 목록]

1. 📝 CareerLogger (이력 자동 입력 컴포넌트)
- 사용자가 입력한 공모전/인턴/활동 경험을 타임라인 형태로 정리
- 입력 필드: 제목, 역할, 기간, 성과(선택), 태그
- 저장 시 AI가 자동으로 클론 정보 업데이트
- UI는 Toss의 폼 입력처럼 부드럽고 질문형 스타일 (ex. “무엇을 하셨나요?”)

2. 📁 AIPortfolioGenerator (포트폴리오 자동 생성)
- CareerLogger에 저장된 데이터를 불러와 카드형 UI로 나열
- 각 카드: 활동 제목, 기간, 요약 성과, 태그 표시
- 포트폴리오 전체를 PDF로 저장하거나 외부 공유 링크 생성
- 포맷은 Toss의 카드 송금 내역처럼 깔끔한 블록형 스타일

3. 🤝 CloneRecommender (팀원 추천)
- AI가 클론 데이터를 분석해 협업 가능한 사용자 추천
- 추천 기준: 목표 일치도, 스킬 매칭, 활동 시간대
- 추천 결과는 카드 슬라이드 형식 (좌우 스와이프 가능)
- 각 카드에 “함께하기” 버튼, “패스” 버튼

4. 🧠 CareerInsightChat (AI 대화형 커리어 피드백)
- 사용자가 AI에게 자신의 이력에 대해 질문하거나 피드백 요청
- Chat UI: 말풍선형, 좌우 정렬
- 분석된 키워드, 강점, 직무 추천 등 시각화 요소 포함 (ex. 그래프, 워드클라우드)
- 사용자의 입력을 분석하여 클론을 정교화

5. 🔍 ExternalSNSConnector (외부 인물 SNS 탐색)
- 사용자가 특정 키워드로 사람을 검색하면 AI가 SNS(LinkedIn, 블로그, 인스타 등) 링크 추천
- 검색 예시: “UX 디자이너 서울 근처”
- 출력은 외부 프로필 썸네일 카드 3~5개 (이름, 직무, 링크 포함)
- UI는 Toss의 알림 리스트처럼 간결하고 아이콘+텍스트 구성

6. 📊 JobMatchingBoard (채용/협업 제안)
- 기업이 등록한 포지션에 유저가 적합한지 AI가 분석 후 추천
- 유저는 리스트를 보고 “지원하기” 또는 “저장하기” 가능
- 기업 대시보드는 클론 기반 추천 리스트와 적합도 점수 제공
- 카드 기반의 채용 공고 리스트 (ex. Toss의 결제 내역 블록 느낌)

7. 💰 TokenRewardSystem (보상 시스템)
- 유저는 활동 시 토큰 획득 → 이력서 첨삭, 포트폴리오 강화 등에 사용
- 리워드 내역: 초대 보상, 활동 보상, 첨삭 소비
- UI는 Toss의 포인트/리워드 대시보드 느낌

[공통 컴포넌트]
- NavigationBar (하단 탭바 or 좌측 사이드바)
  - 탭: ‘내 이력’, ‘추천 팀원’, ‘피드백’, ‘탐색’, ‘내 정보’
- Header: 페이지 상단에 간단한 페이지명과 유저 아이콘
- Button: 파란색(#3182f6), 라운드, hover 시 밝은 블루 (#63a3ff)

[사용자 흐름 예시]
1. 회원가입 및 간단한 온보딩 (MBTI, 직무 관심사 등)
2. 이력 입력 → AI 클론 생성 → 포트폴리오 자동화
3. 팀원 추천 또는 외부 전문가 탐색
4. AI 피드백 받기 / 이력서 생성
5. 토큰 획득 → 부가 기능 사용

[결과물 요구]
- 컴포넌트별 JSX + Tailwind 적용된 UI 코드
- 상태관리 구조(Zustand or Context)
- 각 기능별 분리된 폴더 구조 제안
- Framer Motion 기반 페이지 전환 로직 포함
